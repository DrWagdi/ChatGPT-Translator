# ChatGPT-Translator
這是一個使用ChatGPT API開發的翻譯插件，可以自動判斷輸入的語言並翻譯成多種語言。
This is a translation plugin developed using the ChatGPT API, which can automatically detect the input language and translate it into multiple languages.
